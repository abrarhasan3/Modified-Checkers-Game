# Modified-Checkers-Game
This a modified Version of checkers Game. Made for Academic purpose. This is our project for AI laboratory.

## Prequisite 
<ol>
  <li><b>Python 3.9.12 </b></li>
  <li><b>Pygame 2.3.0</b></li>
</ol>

<ol>

  <li><b>Black (Human)</b></li>
